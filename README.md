# full_stack_TODO_with_NEXTJS

## これを通して学べること
・Next.jsとNode .js (Express)とTypeScriptでフルスタックTodoアプリケーションが作れる
・AWS AmplifyとAppRunnerを使ってデプロイできる
・デプロイ時における環境変数の取り扱い
・CRUD操作できるPrismaを利用したAPIが作れる
